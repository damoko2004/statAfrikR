## 🌱 Comment contribuer ?

1.  Fork le dépôt sur [GitHub](https://github.com/damoko2004/statAfrikR)
2.  Crée une branche : `git checkout -b feature/ma-contribution`
3.  Commit tes changements : `git commit -m "feat: description"`
4.  Push et ouvre une Pull Request
